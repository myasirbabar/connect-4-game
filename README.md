# Connect-4-Game
Its a two player Connect 4 Game Build On C++ Console

---------------
How to Begin
- Open the main.cpp file in your editor. 
- Compile and run
- Happy Execution :)

How To Play
- Each player has to take its turn.
- The One player who is able to make a 4 consective sequence in any direction wins.
