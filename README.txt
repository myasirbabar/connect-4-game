Its a two player game build on C++ Console using OOP Concepts. 

How to Begin
	1) Open the main.cpp file in your editor. 
	2) Compile and run
	3) Happy Execution :)

How To Play
	1) Each player has to take its turn.
	2) The One player who is able to make a 4 consective sequence in any direction wins.
